# Agent Instructions

This document provides instructions for AI agents working with this codebase.

## Code Style and Linting

This project uses `ruff` for code formatting and linting. Before submitting any changes, please ensure that you have run the following commands from the root of the repository:

```bash
uv run ruff check --fix .
uv run ruff format .
```

This will automatically fix most linting issues and ensure that your code is formatted correctly.
